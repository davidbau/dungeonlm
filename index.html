dungeonlm

MIT Dungeon (1980 — the original Zork) in your browser, with an optional WebGPU-powered LLM that translates plain English into the game's canonical parser vocabulary.
LLM parser: off
Classic commands (take egg, n, read leaflet) always work. When the LLM is on, natural-language phrasing like "please grab the shiny egg" gets translated before the parser sees it. Requires a WebGPU-capable browser; first enable triggers a ~850 MB model download that the browser will cache.
source · design doc · MIT Dungeon by Anderson, Blank, Daniels, Lebling (MIT LCS, 1977—1980).
